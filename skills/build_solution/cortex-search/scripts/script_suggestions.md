# Script Suggestions

Potential automation scripts for future implementation:

## Deployment Scripts

- **deploy_search.sh**: Full pipeline: stage → parse → index
- **refresh_documents.sh**: Update documents and refresh service

## Testing Scripts

- **test_search.py**: Run sample queries and verify results
- **validate_chunks.py**: Check chunk sizes and quality

## Monitoring Scripts

- **check_indexing.py**: Verify service indexing status
- **usage_report.py**: Query usage history views
